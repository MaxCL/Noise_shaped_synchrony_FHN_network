# Noise_shaped_synchrony_FHN_network
This repository contains the data and the code for the figures presented in "Noise-shaped Synchrony in Neuronal Oscillator Networks" submitted for publication to Phys. Rev. Lett. 

Recommended use: Python virtual environment.

Otherwise (system wide):

Install requirements:
- jupyterlab
- python3-notebook
- python3-jupyterlab-server
- jupyter-server
